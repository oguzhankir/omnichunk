# Roadmap

## Near term (P0)

- Statement-safe oversized splitting for code
- Real streaming API (incremental generation)
- Shared byte/line indexing utilities (remove repeated full scans)
- Overlap contextualization correctness improvements
- Rule-sync CI enforcement

## Mid term (P1)

- Language-specific tree-sitter query packs
- Scope-level sibling accuracy improvements
- Better import usage filtering from AST symbol usage
- Markdown fenced-code routing through code engine
- Benchmark suite against baseline chunkers

## Longer term (P2)

- Plugin API for custom parsers/formatters
- Performance dashboard and regression gates
- Extended language and mixed-format coverage
