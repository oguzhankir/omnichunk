# v0.10.1 — Stable API Pre-release

Omnichunk 0.10.1 focuses on **stability signaling**, **documentation**, and a new **proposition extraction** API, while keeping the core chunking engine unchanged.

## Highlights

- **MkDocs Material documentation** (`omnichunk[docs]`), including API stability policy, migration notes, and performance methodology.
- **`py.typed`** marker for PEP 561 type checking.
- **`Chunker.extract_propositions(...)`** with `mode="heuristic"` (stdlib-only patterns) or `mode="llm"` (user-supplied JSON-producing `llm_fn`).
- **`omnichunk[all]`** meta-extra: most optional integrations without pulling in heavy ML tooling by default.

## Install

```bash
pip install omnichunk==0.10.1
pip install "omnichunk[all]==0.10.1"
pip install "omnichunk[docs]==0.10.1"
```
